# Secret Managment

> OneOps and Keywhiz


## What are Secrets?

Anything that should not be public.

- username / password
- ssh keys
- certificates
- keystores
- ...


## Practices to Avoid

- No credentials
- Default credentials
- No secrets
- Unsecured secrets
- Obscured secrets
- Your own secret management

Note:
- security is hard
- but dont avoid the efforts
- breaches are worse
- dont do it yourself, you will fail
